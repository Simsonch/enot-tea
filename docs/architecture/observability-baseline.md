# Базовая наблюдаемость (MVP)

## Логирование

- Логировать структурированный JSON в API.
- Включать request id, route, status code и latency.
- Никогда не логировать секреты, auth-токены и персональные данные.

## Мониторинг

- Отслеживать доступность health endpoint API.
- Отслеживать долю успешного создания заказов.
- Отслеживать количество ошибок приложения.

## Алертинг

- Поднимать SEV-1 алерт при повторяющихся падениях health API.
- Поднимать алерт при падении доли успешного создания заказов ниже порога.
